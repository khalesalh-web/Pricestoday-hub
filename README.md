PricesToday Hub

مرحبًا بك في PricesToday Hub – موقع شامل يعرض أسعار العملات، العملات الرقمية، المعادن، والمواد الأولية بشكل مباشر.

الصفحات:

- الرئيسية: index.html
- المواد الأولية: materials.html (القمح، السكر، القهوة، الذرة، الصويا، القطن، النفط، الغاز)
- المعادن: metals.html (الذهب، الفضة، البلاتين، النحاس، النفط، الغاز)
- العملات النقدية: fiat.html (محدثة تلقائيًا مقابل الدولار)
- العملات الرقمية: crypto.html (Bitcoin, Ethereum, Binance Coin, Cardano, Solana، محدثة تلقائيًا)
- المحوّل: converter.html (لتحويل بين العملات، العملات الرقمية، المعادن، والمواد الأولية)

ميزات الموقع:

- تحديث تلقائي لكل الأسعار كل دقيقة
- واجهة ثنائية اللغة: عربي / إنجليزي
- تصميم مستجيب للهاتف والكمبيوتر
- أسعار المواد والمعادن حاليا تجريبية (يمكن ربط API لاحقًا)
- العملات الرقمية والنقدية حقيقية ومحدثة من APIs مجانية

ملاحظات:

- يمكنك تجربة المحوّل للتحويل بين أي عملة أو معدن أو مادة أولية
- يمكن إضافة المزيد من العملات أو المواد لاحقًا بسهولة
- التصميم بسيط ويمكن تحسينه ليصبح أكثر جذبًا واحترافية

تعليمات الاستخدام:

1. افتح أي صفحة من الصفحات عبر GitHub Pages
2. تأكد من الاتصال بالإنترنت لمشاهدة الأسعار الحية للعملات والعملات الرقمية
3. استخدم المحوّل للتحويل بين العناصر المختلفة
